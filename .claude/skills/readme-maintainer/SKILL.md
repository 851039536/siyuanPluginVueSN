# README 维护助手

这个Skill帮助您根据当前页面功能或项目状态来维护README.md文件。无论您是开始一个新项目，还是需要更新现有项目的文档，这个工具都能提供系统化的指导。

## 核心功能

1. **项目结构分析** - 分析当前项目的目录结构和文件组织
2. **当前页面功能检查** - 检查当前页面或功能模块的核心要素
3. **README模板生成** - 根据分析结果生成合适的README模板
