# README模板生成工作流

这个工作流根据项目分析结果生成合适的README.md模板。

## 模板生成步骤

### 1. 选择模板类型 ⭐⭐⭐⭐⭐
- [ ] **基础项目** - 简单的库或工具
- [ ] **Web应用** - 完整的Web应用程序
- [ ] **API服务** - RESTful API或微服务
- [ ] **CLI工具** - 命令行工具
- [ ] **移动应用** - 移动端应用
- [ ] **桌面应用** - 桌面应用程序

### 2. 配置基本信息 ⭐⭐⭐⭐⭐
- [ ] 项目名称和徽章
- [ ] 简短描述（1-2句话）
- [ ] 详细描述（功能、特点、优势）
- [ ] 演示链接或截图

### 3. 核心内容生成 ⭐⭐⭐⭐⭐
- [ ] **快速开始** - 最简安装和使用步骤
- [ ] **使用示例** - 基础到高级的使用示例
- [ ] **配置说明** - 所有配置选项


## 模板结构

# [项目名称]

[简短描述]
