## reloadUI 第二次修复（401 Unauthorized）
- 问题：第一次修复后，reloadUI/reloadFiletree/reloadTag 仍报 401 Unauthorized。根因：这 5 个 reload 函数使用原生 fetch() 而非 fetchSyncPost()，缺少认证 token
- 修复：5 个 reload 函数全部从原生 fetch(url, {method, headers, body}) 改为 fetchSyncPost(url, data)
  - reloadUI → fetchSyncPost("/api/system/reloadUI", {})
  - reloadFiletree → fetchSyncPost("/api/ui/reloadFiletree", {})
  - reloadTag → fetchSyncPost("/api/ui/reloadTag", {})
  - reloadAttributeView → fetchSyncPost("/api/ui/reloadAttributeView", { id: blockId })
  - reloadProtyle → fetchSyncPost("/api/ui/reloadProtyle", { id: blockId })
- 验证通过：ESLint 0 error
