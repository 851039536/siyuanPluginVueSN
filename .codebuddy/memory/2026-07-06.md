# dataBackup → s3Backup 合并完成

## 日期：2026-07-06

### 变更概述
将 `src/features/dataBackup`（本地 ZIP 压缩备份 + 自动备份定时器）完全迁移合并到 `src/features/s3Backup` 模块中，dataBackup 模块已彻底删除。

### 具体变更

#### s3Backup 模块增强（7 个文件修改）
- **modules/BackupManager.ts** — 合并版备份管理器：保留文件扫描 + 新增 ZIP 打包（performFullBackup）/ 本地备份管理（scanBackupDir/deleteBackupFile），统一 scanDirectory 实现
- **types/index.ts** — 新增 BackupMode / AutoBackupSettings / LocalBackupInfo 接口，S3BackupStorage 新增 autoBackupSettings 和 backupHistory 存储槽
- **index.ts** — S3Backup 类增强：新增自动备份定时器（分钟/小时/天）、autoBackupTrigger 事件处理、持久化 Modal 模式
- **index.vue** — 主面板重构：新增备份模式选择区（多选复选框）、自动备份设置区、本地备份历史列表，手动备份根据 backupMode 分发
- **composables/useS3Backup.ts** — phaseLabel 扩展 packing/compressing/saving
- **styles/index.scss** — 新增备份模式选择器、自动备份设置区样式
- **i18n** — zh_CN + en_US 各从 54 → 78 keys（新增 24 个本地备份专属 key）

### 合并后的功能
- 备份模式：可多选「本地 ZIP」+「S3 上传」，灵活组合
- 手动备份：一键触发，顺序执行所选模式
- 自动备份：支持每分钟/每小时/每天，可设置时间 + 保留份数
- 本地备份历史：展示 data-backup/ 目录下 ZIP 文件，支持删除
- S3 云端备份列表：下载/恢复/删除（功能不变）
