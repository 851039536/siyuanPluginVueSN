# AI内容生成器 - 编辑模式使用指南

## 功能概述

AI内容生成器现在支持**编辑模式**，允许你直接编辑思源笔记中的文档内容，并提供差异对比和撤回功能。

## 主要功能

### 1. 编辑模式切换
- 点击顶部工具栏的**编辑图标**（铅笔图标）切换编辑模式
- 启用后会显示蓝色高亮

### 2. 选择目标文档
- 在编辑模式下，点击"选择要编辑的文档"按钮
- 系统会自动获取当前光标所在的文档
- 文档内容会自动加载到编辑区域

### 3. 编辑内容
编辑模式提供两种视图：

#### 编辑视图（默认）
- 左侧：Markdown编辑器，可直接编辑内容
- 右侧：实时预览，查看渲染效果
- 支持标准Markdown语法

#### 差异对比视图
- 点击"查看差异"按钮切换
- 左侧：原始文档内容
- 右侧：修改后的内容
- 便于对比修改前后的变化

### 4. 应用编辑
- 点击"应用"按钮将修改保存到思源笔记
- 使用 `updateBlock` API 更新文档内容
- 系统会保存编辑历史用于撤回

### 5. 撤回编辑
- 应用编辑后会显示"撤回"按钮
- 点击可恢复到修改前的状态
- 仅保留最近一次编辑的历史

## 使用场景

### 场景1：AI辅助文档修订
1. 启用编辑模式
2. 选择要修改的文档
3. 使用AI生成改进建议
4. 在编辑器中应用修改
5. 查看差异对比
6. 应用到文档

### 场景2：批量内容优化
1. 选择文档
2. 提示AI："请优化这篇文档的格式和结构"
3. AI生成优化后的内容
4. 对比差异
5. 确认无误后应用

### 场景3：内容翻译
1. 选择中文文档
2. 提示AI："将这篇文档翻译成英文"
3. 查看翻译结果
4. 应用更新

## 技术实现

### 核心API
- `getBlockByID()`: 获取文档块信息
- `exportMdContent()`: 获取文档Markdown内容
- `updateBlock()`: 更新文档内容

### 状态管理
```typescript
- editMode: 是否启用编辑模式
- editTargetDoc: 目标文档信息
- originalContent: 原始文档内容
- generatedContent: 编辑后的内容
- showDiff: 是否显示差异对比
- hasContentChanged: 内容是否已修改
- lastEditHistory: 编辑历史（用于撤回）
```

### 编辑流程
```
1. 用户启用编辑模式
2. 选择目标文档
3. 加载文档内容到编辑器
4. 用户编辑或使用AI生成新内容
5. 查看差异对比（可选）
6. 应用编辑到文档
7. 保存编辑历史
8. 可选：撤回编辑
```

## 注意事项

1. **自动保存**: 编辑不会自动保存，需要点击"应用"按钮
2. **撤回限制**: 仅支持撤回最近一次编辑
3. **内容同步**: 应用编辑后，思源笔记会自动刷新文档显示
4. **权限检查**: 确保对目标文档有编辑权限

## 快捷键

- `Ctrl+Enter`: 生成内容（普通模式）
- 暂无编辑模式专用快捷键

## 样式说明

### 差异对比视图
- 使用栅格布局，左右对比
- 中间分隔线区分原始内容和新内容
- 支持滚动查看长内容

### 编辑视图
- 左侧编辑器使用等宽字体
- 右侧实时渲染Markdown
- 支持响应式布局
