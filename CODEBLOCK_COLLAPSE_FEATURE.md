# 代码块折叠功能测试指南

## 功能概述

新增的代码块折叠功能允许用户自动折叠超出指定高度的代码块，提供更好的阅读体验。

## 功能特性

### 1. 折叠开关
- 可以启用或禁用代码块折叠功能
- 默认启用
- 切换后立即生效

### 2. 折叠高度设置
- 可调节折叠触发的像素高度
- 范围：200px - 800px
- 步长：50px
- 默认值：400px

### 3. 智能折叠
- 仅当代码块内容超出设定高度时才显示折叠按钮
- 编辑时自动展开代码块
- 支持移动端适配

## 使用方法

### 启用折叠功能

1. 打开思源笔记设置
2. 进入"代码块美化"设置页面
3. 在"代码块折叠"部分启用开关
4. 调节折叠高度滑块设置触发高度

### 折叠交互

- 当代码块高度超过设定值时，底部会显示展开按钮
- 点击展开按钮可完全展开代码块
- 在代码块内编辑时自动展开
- 滚动页面时动态检测和添加折叠按钮

## 技术实现

### 样式注入
```css
.code-block .code-down {
  width: 100%;
  text-align: center;
  position: absolute;
  bottom: 0px;
  left: 0;
  padding: 70px 0px 10px;
  background-image: linear-gradient(to bottom, rgba(255, 255, 255, 0), rgba(255, 255, 255, 1));
}
```

### JavaScript 功能
- 自动检测代码块高度
- 动态添加展开按钮
- 监听滚动事件和DOM变化
- 编辑状态检测

### 主题适配
- 支持浅色和深色主题
- 渐变背景自动适配
- 按钮样式响应主题变化

## 预览效果

设置页面包含实时预览功能：
- 显示当前设定的折叠高度
- 演示代码块折叠效果
- 滑块调节时实时更新预览

## 兼容性

- 支持桌面端和移动端
- 兼容思源笔记所有主题
- 与现有代码块美化功能完全兼容
- 不影响代码块的原有功能

## 故障排除

### 折叠功能不生效
1. 检查是否启用了折叠开关
2. 确认代码块高度是否超过设定值
3. 刷新页面重新加载功能

### 按钮样式异常
1. 检查主题设置
2. 确认CSS样式是否正确加载
3. 清除浏览器缓存

### 性能问题
1. 折叠功能使用防抖处理，避免频繁操作
2. 仅在必要时添加事件监听器
3. 动态管理DOM元素，避免内存泄漏
