# 列表设置功能

## 功能概述

新增的列表设置功能为思源笔记插件提供了强大的自定义列表样式功能，支持无序列表和有序列表的个性化配置。

## 主要特性

### 🎯 无序列表自定义
- **多层级符号配置**: 支持三级嵌套列表的不同符号设置
- **符号选择**: 提供多种预设符号（▪, •, ◦, ▸, ▹, ▶, ◉, ○, ◇, ◆）
- **自定义符号**: 支持用户输入自定义符号
- **样式调整**: 可调节符号大小和左边距

### 🔢 有序列表自定义
- **多种编号格式**: 支持十进制、带零十进制、字母大小写、罗马数字等格式
- **标准CSS规范**: 使用标准CSS `list-style-type` 属性

### 🌐 应用范围
- **列表块**: 应用于文档中的普通列表块
- **嵌入块**: 应用于嵌入块内的列表
- **浮窗**: 应用于浮窗中的列表

## 技术实现

### CSS生成策略
```css
/* 无序列表块示例 */
[data-subtype="u"]>.li[data-subtype="u"]>.protyle-action svg {
    color: transparent;
}

[data-subtype="u"]>.li[data-subtype="u"]>.protyle-action::before {
    font-size: 1.5em;
    line-height: 1;
    margin-bottom: 0px;
    margin-left: 13px;
    font-family: Arial;
    content: "▪";
}

[data-subtype="u"]>[data-subtype="u"] .li[data-subtype="u"]>.protyle-action::before {
    content: "•";
}

[data-subtype="u"]>[data-subtype="u"] .li[data-subtype="u"] .li[data-subtype="u"]>.protyle-action::before {
    content: "◦";
}
```

### 数据持久化
- 使用 localStorage 存储设置
- 自动保存用户配置
- 支持设置重置功能

## 使用方法

1. **打开设置面板**: 点击思源笔记右侧边栏的设置图标
2. **选择列表设置**: 在左侧导航中选择"列表设置"
3. **启用自定义样式**: 开启"启用自定义样式"开关
4. **配置符号**: 选择或输入各级列表的符号
5. **调整参数**: 设置符号大小和边距
6. **选择应用范围**: 决定样式应用的具体场景

## 配置选项详解

### 无序列表符号
- 一级列表: 通常使用较大的实心符号 (▪)
- 二级列表: 使用中等大小的符号 (•)
- 三级列表: 使用较小的符号 (◦)

### 有序列表格式
- `decimal`: 1. 2. 3.
- `decimal-leading-zero`: 01. 02. 03.
- `lower-alpha`: a. b. c.
- `upper-alpha`: A. B. C.
- `lower-roman`: i. ii. iii.
- `upper-roman`: I. II. III.

### 样式参数
- **符号大小**: 0.8em - 2.5em
- **左边距**: 0px - 30px

## 注意事项

1. 自定义符号建议使用单个字符或 Unicode 符号
2. 符号大小会影响列表的整体视觉效果
3. 不同主题下可能需要调整符号颜色
4. 设置会自动保存，无需手动确认
