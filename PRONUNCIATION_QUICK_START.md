# 谐音翻译功能 - 快速开始

## 一分钟上手

### 1. 配置API（首次使用）

1. 点击右上角**超级面板**图标
2. 点击**设置**图标（齿轮）
3. 选择API供应商（推荐：通义千问）
4. 输入API密钥
5. 配置自动保存 ✓

### 2. 使用谐音翻译

**方法一：右键菜单**
```
选中单词 → 右键 → 🔊 谐音翻译 → 自动复制到剪贴板
```

**方法二：快捷键**
```
选中单词 → Shift+Ctrl+H (Windows) / Shift+Cmd+H (Mac)
```

### 3. 示例

选中单词 `system`，右键选择"谐音翻译"，得到：

```markdown
#### system

单词：system
音标：[ˈsɪstəm]
释义：系统；体系；制度
谐音：西斯腾(xī sī téng)
发音：['sɪs-təm] 两个音节，重音在第一音节
```

## 常见问题

**Q: 提示"请先在超级面板中配置API密钥"？**
A: 打开超级面板 → 设置 → 输入API密钥

**Q: 生成失败？**
A: 检查：
1. API密钥是否正确
2. 网络连接是否正常
3. API额度是否充足

**Q: 只能生成英文单词的谐音吗？**
A: 是的，目前只支持英文单词

**Q: 可以批量生成吗？**
A: 暂不支持，需要逐个生成

## 技术说明

- **配置存储**：使用思源API保存，重启后保持
- **API调用**：使用超级面板的统一配置
- **结果处理**：自动复制到剪贴板
- **无侵入性**：不占用侧边栏空间

## 启用/禁用

在超级面板中找到"谐音翻译"卡片，点击开关即可启用或禁用功能。
