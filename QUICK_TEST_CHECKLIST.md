## 常见问题快速排查

### 问题：没有流式输出
**检查**：
```javascript
// 控制台应该显示：
开始生成内容，配置: {provider: "deepseek", model: "deepseek-chat", hasOnChunk: true}
```
如果 `hasOnChunk: false`，说明回调没有正确传递

### 问题：提示词不生效
**检查**：
```javascript
// 控制台应该显示：
系统提示词: [你设置的提示词内容]
```
如果显示的是默认提示词，说明提示词没有正确加载

### 问题：文档内容没有使用
**检查**：
```javascript
// 控制台应该显示：
已添加文档上下文，标题: XXX, 内容长度: XXX
构建完整提示词，包含上下文，长度: XXX
```
如果没有这些日志，说明文档引用失败
