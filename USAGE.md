# 页面锁定功能使用说明

## 功能介绍

页面锁定功能允许你对指定的文档进行加密保护，只有输入正确密码才能查看内容。

## 如何使用

### 页面顶部按钮（唯一入口）

1. 打开需要锁定的文档
2. 在**文档顶部右侧**会显示 **🔒 锁定页面** 或 **🔓 解锁页面** 按钮
3. 点击按钮后输入密码并确认
4. 页面锁定/解锁成功

## 解锁页面

### 临时解锁（本次会话有效）

1. 打开已锁定的文档，会显示锁定遮罩
2. 点击“解锁页面”按钮
3. 输入密码
4. 解锁成功后可以查看内容（本次会话有效）

### 永久解锁（移除锁定）

1. 打开已锁定的文档
2. 点击页面顶部的 **🔓 解锁页面** 按钮
3. 输入密码确认
4. 页面锁定被永久移除

## 注意事项

- 密码使用SHA-256加密存储，安全可靠
- 忘记密码后无法恢复，请妥善保管密码
- 每个文档可以设置不同的密码
- 数据存储在插件本地，不会上传到云端
- 解锁后，当前会话内可以正常访问，关闭思源后需重新解锁

## 测试步骤

1. **启动开发环境**
   ```bash
   pnpm dev
   ```

2. **在思源笔记中启用插件**
   - 设置 → 集市 → 下载 → 本地插件
   - 找到插件并启用

3. **测试锁定功能**
   - 创建一个测试文档
   - 点击文档顶部右侧的“锁定页面”按钮
   - 输入测试密码（如：123456）

4. **测试解锁功能**
   - 关闭并重新打开该文档
   - 应该看到锁定遮罩
   - 输入密码解锁查看内容

5. **测试永久解锁**
   - 点击页面顶部的“解锁页面”按钮
   - 输入密码确认
   - 页面应该恢复正常状态
