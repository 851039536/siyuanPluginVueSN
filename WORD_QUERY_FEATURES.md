# 词典查询功能增强说明

## 🎉 新增功能概览

本次更新为词典查询功能添加了多个实用的增强特性，极大提升了单词学习和管理的体验。

## ✨ 主要新功能

### 1. 📚 查询历史记录
- **自动记录**：每次查询的单词自动保存到历史记录
- **快速访问**：点击历史记录中的单词可快速查看之前的查询结果
- **时间显示**：显示查询时间（刚刚、X分钟前、X小时前、X天前）
- **一键清除**：支持清除所有历史记录
- **智能存储**：最多保存50条历史记录，超出自动删除最旧的记录
- **去重机制**：相同单词的重复查询会更新时间戳并移至顶部

**使用方式**：
- 点击顶部工具栏的 🕒 图标打开历史记录面板
- 点击任意历史记录项快速查看该单词

### 2. ⭐ 单词收藏功能
- **收藏管理**：将重要单词添加到收藏列表
- **持久化存储**：收藏的单词永久保存在本地
- **快速标记**：查询结果区域提供收藏/取消收藏按钮
- **独立面板**：专门的收藏面板展示所有收藏的单词
- **删除管理**：支持单独删除收藏项或清空所有收藏

**使用方式**：
- 查询单词后，点击结果下方的 ⭐ 收藏按钮
- 点击顶部工具栏的 ⭐ 图标打开收藏面板
- 在收藏面板中点击单词快速查看，点击删除图标移除收藏

### 3. 🔊 语音播放功能
- **TTS发音**：使用浏览器内置的语音合成技术播放单词发音
- **英美发音**：支持英式(UK)和美式(US)发音切换
- **自动播放**：可选择查询后自动播放发音
- **手动播放**：结果区域提供播放按钮随时重听

**使用方式**：
- 点击结果下方的 🔊 播放按钮手动播放
- 在高级选项中开启"自动播放"功能

### 4. ⚙️ 高级选项设置
- **发音偏好**：选择英式或美式发音
- **自动播放**：开关自动播放发音功能
- **相关词推荐**：控制是否显示相关词（预留功能）
- **设置持久化**：所有设置自动保存到本地存储

**使用方式**：
- 点击顶部工具栏的 ⚙️ 图标打开高级选项面板
- 调整各项设置，自动保存无需手动确认

### 5. 📤 导出到思源笔记
- **一键导出**：将查询结果导出为Markdown格式
- **复制到剪贴板**：导出内容自动复制，可直接粘贴到文档
- **格式化内容**：包含单词标题和完整的查询结果
- **快速记录**：方便将学习的单词保存到笔记中

**使用方式**：
- 点击结果下方的 📤 导出按钮
- 在思源笔记中任意位置粘贴（Ctrl+V）

### 6. 📋 增强的复制功能
原有的复制功能保持不变，支持：
- 复制全部内容
- 复制音标/拼音
- 复制释义
- 复制英文翻译（中文查询时）
- 复制谐音
- 复制例句

## 💾 数据存储

所有数据使用 `localStorage` 存储在浏览器本地：

| 存储键名 | 数据内容 | 说明 |
|---------|---------|------|
| `word-query-history` | 查询历史记录 | JSON数组，最多50条 |
| `word-query-favorites` | 收藏列表 | JSON数组，无数量限制 |
| `word-query-pronunciation-type` | 发音类型 | uk 或 us |
| `word-query-auto-play` | 自动播放开关 | true 或 false |
| `word-query-show-related` | 显示相关词 | true 或 false |
| `word-query-api-key-{provider}` | API密钥 | 按供应商分别存储 |
| `word-query-api-provider` | API供应商 | tongyi/openai/deepseek/custom |
| `word-query-custom-endpoint` | 自定义API端点 | URL字符串 |

## 🔄 数据管理

### 历史记录管理
- 自动去重：相同单词只保留最新一条
- 智能排序：最新查询的单词排在最前面
- 容量限制：超过50条自动删除最旧记录
- 一键清除：提供清除所有历史的功能

### 收藏管理
- 独立存储：与历史记录分开管理
- 永久保存：不会自动删除
- 灵活操作：支持单个删除或全部清空
- 状态同步：收藏状态实时更新UI

3. 复制需要的部分（音标、释义等）

## ⚡ 性能优化

- **本地存储**：所有数据存储在本地，无需网络请求
- **懒加载**：面板按需显示，减少初始渲染

- **智能缓存**：历史记录作为天然缓存，快速查看已查询单词

## 💡 使用建议

1. **合理使用收藏功能**：只收藏真正重要或难记的单词，避免收藏过多
2. **定期清理历史**：历史记录会自动限制数量，但可以手动清理
3. **设置发音偏好**：根据学习目标选择英式或美式发音
4. **结合笔记使用**：将重要单词导出到思源笔记，建立个人词汇库
5. **充分利用快捷键**：使用 Ctrl+Enter 快速查询，Escape 清除结果
