# 词典查询功能测试指南

## 测试环境准备

1. 确保思源笔记正在运行
2. 插件已正确安装并启用
3. 浏览器支持 Web Speech API（Chrome、Edge等主流浏览器）

## 功能测试清单

### ✅ 1. 历史记录功能测试

**测试步骤：**
1. 打开词典查询面板
2. 依次查询几个单词：hello, world, test, example
3. 点击顶部 🕒 图标打开历史记录面板
4. 验证：
   - [ ] 所有查询过的单词都显示在历史中
   - [ ] 最新查询的单词在最上面
   - [ ] 时间显示正确（刚刚、X分钟前等）
5. 点击历史记录中的某个单词
6. 验证：
   - [ ] 该单词的查询结果正确显示
   - [ ] 历史面板自动关闭
7. 再次查询之前查过的单词（如 hello）
8. 验证：
   - [ ] 历史记录中没有重复项
   - [ ] hello 移动到历史记录顶部
9. 点击"清除"按钮
10. 验证：
    - [ ] 所有历史记录被清空
    - [ ] 显示"暂无查询历史"提示

### ✅ 2. 收藏功能测试

**测试步骤：**
1. 查询单词 "computer"
2. 点击结果下方的 ⭐ 收藏按钮
3. 验证：
   - [ ] 按钮变为橙色（已收藏状态）
   - [ ] 显示提示"已添加到收藏"
4. 点击顶部 ⭐ 图标打开收藏面板
5. 验证：
   - [ ] "computer" 显示在收藏列表中
   - [ ] 收藏时间正确显示
6. 再次点击收藏按钮
7. 验证：
   - [ ] 按钮变回金色（未收藏状态）
   - [ ] 显示提示"已取消收藏"
   - [ ] 收藏面板中该单词被移除
8. 收藏多个单词
9. 在收藏面板中点击某个单词
10. 验证：
    - [ ] 该单词的查询结果正确显示
    - [ ] 收藏状态显示为已收藏
11. 点击收藏面板的删除图标
12. 验证：
    - [ ] 该单词从收藏中移除
13. 点击"清除"按钮
14. 验证：
    - [ ] 所有收藏被清空
    - [ ] 显示"暂无收藏单词"提示

### ✅ 3. 语音播放功能测试

**测试步骤：**
1. 查询英文单词 "pronunciation"
2. 点击结果下方的 🔊 播放按钮
3. 验证：
   - [ ] 浏览器播放单词发音
   - [ ] 发音清晰可听
4. 打开高级选项面板
5. 切换发音类型为"美式发音"
6. 再次点击播放按钮
7. 验证：
   - [ ] 发音变为美式口音（如果浏览器支持）
8. 开启"自动播放"选项
9. 查询新单词 "technology"
10. 验证：
    - [ ] 查询完成后自动播放发音
11. 关闭"自动播放"选项
12. 查询新单词
13. 验证：
    - [ ] 不会自动播放，需手动点击

### ✅ 4. 高级选项测试

**测试步骤：**
1. 点击顶部 ⚙️ 图标（右边第二个）打开高级选项
2. 验证面板内容：
   - [ ] 显示发音设置选项（英式/美式）
   - [ ] 显示自动播放开关
   - [ ] 显示相关词推荐开关
3. 切换各项设置
4. 关闭面板后重新打开
5. 验证：
   - [ ] 所有设置保持之前的状态（持久化成功）
6. 刷新页面
7. 打开高级选项
8. 验证：
   - [ ] 设置依然保持（localStorage 存储成功）

### ✅ 5. 导出功能测试

### ✅ 6. 复制功能测试

## 已知限制

1. **语音播放质量**：依赖浏览器内置TTS引擎，不同浏览器效果可能不同
2. **存储容量**：localStorage有5MB限制，但对于文本数据完全足够
3. **离线使用**：历史和收藏可离线使用，但查询新单词需要网络
4. **跨设备同步**：暂不支持，计划未来版本添加

## 快速测试场景

如果时间有限，可以进行以下快速测试：

1. **基础流程**（5分钟）
   - 查询一个单词
   - 收藏该单词
   - 打开历史面板查看
   - 播放发音
   - 导出到笔记

2. **持久化测试**（2分钟）
   - 收藏几个单词
   - 关闭浏览器
   - 重新打开
   - 验证数据保留

3. **UI测试**（3分钟）
   - 打开所有面板
   - 测试所有按钮
   - 观察动画效果
