# 代码块美化功能使用指南

## 功能概述

代码块美化功能允许你为思源笔记中的代码块选择不同的显示风格，让代码展示更加美观和专业。

## UI 特性

### 视觉增强
- ✨ **卡片式选择器** - 直观的风格选择界面
- 🎨 **语法高亮预览** - 实时查看代码高亮效果
- 🔢 **行号显示** - 清晰的代码行标识
- 🎭 **悬停效果** - 流畅的交互动画
- 🌈 **渐变装饰** - 精美的视觉细节

### 交互体验
- 点击卡片即可切换风格
- 悬停查看动态效果
- 选中状态带有动画反馈
- Mac 按钮支持交互动画

## 支持的风格

### 1. 默认风格 (Default Style)
- 保持思源笔记原有的代码块样式
- 简洁实用，适合日常使用
- 自动适配当前主题

### 2. GitHub 风格 (GitHub Style)
- 模仿 GitHub 代码展示风格
- 特点：
  - 自适应背景色（跟随主题）
  - 清晰的边框分隔
  - 顶部显示语言标签
  - 轻微的阴影效果
- **深色主题适配**：使用思源主题变量，完美融入深色环境

### 3. Mac 风格 (Mac Style)
- 模仿 macOS 终端窗口风格
- 特点：
  - 顶部有红黄绿三个按钮
  - 渐变的标题栏
  - 居中显示语言名称
  - 增强的阴影效果
- **深色主题适配**：标题栏和背景色自动调整，保持视觉一致性

### 4. 卡通风格 (Cartoon Style) 🎨
- 活泼可爱的卡通风格
- 特点：
  - 彩虹渐变边框
  - 跳动的彩色圆点装饰
  - 渐变文字效果
  - 闪烁的装饰元素
  - Comic Sans 字体
  - 悬停动画效果
- **适合场景**：轻松愉快的学习笔记、创意项目文档

## 使用方法

### 1. 打开设置面板
- 点击右侧边栏的"通用设置"图标
- 或通过超级面板进入通用设置

### 2. 选择代码块美化
- 在通用设置面板中，点击"代码块美化"选项卡
- 图标：💻

### 3. 选择风格
- 从风格卡片中选择你喜欢的代码块风格：
  - 📄 默认风格
  - 🐙 GitHub 风格
  - 🍎 Mac 风格
  - 🎨 卡通风格
- 点击卡片即可切换风格
- 选中的风格会显示勾选标记

### 4. 调整高级设置
- **字体大小**：10px - 20px，默认 13px
  - 使用滑块调整代码字体大小
  - 实时预览效果
- **代码间距**：8px - 32px，默认 16px
  - 调整代码块内部的上下间距
  - 让代码更紧凑或更宽松

### 5. 预览效果
- 设置面板会实时显示预览效果
- 预览包含语法高亮和行号
- 可以看到不同风格的代码块展示样式
- 鼠标悬停可查看交互效果
- 字体大小和间距调整会实时反映在预览中

### 6. 保存设置
- 点击"保存"按钮应用设置
- 设置会立即生效，并在下次启动时自动加载
- 所有设置（风格、字体大小、间距）都会被保存

## 技术实现

### 样式应用机制
- 通过在 `<body>` 标签上添加 CSS 类来切换风格
- 类名格式：`codeblock-style-{style}`
- 例如：`codeblock-style-github`

### 持久化存储
- 设置保存在 localStorage 中
- 键名：`general-codeblock-settings`
- 插件加载时自动读取并应用

### 样式定义
- 所有样式定义在 `src/index.scss` 文件中
- 使用 CSS 选择器针对思源的代码块元素
- 支持编辑模式和阅读模式

## 自定义扩展

如果你想添加自己的代码块风格：

1. 在 `src/index.scss` 中添加新的样式规则
2. 在 `CodeBlockSettings.vue` 中添加新的选项
3. 更新国际化文件 (`en_US.json` 和 `zh_CN.json`)

示例：
```scss
/* 自定义风格 */
body.codeblock-style-custom {
  .protyle-wysiwyg [data-node-id][data-type="NodeCodeBlock"] {
    /* 你的自定义样式 */
  }
}
```

## 主题适配

### 深色主题优化
所有代码块风格都已针对深色主题进行优化：

- **使用 CSS 变量**：所有颜色使用思源的主题变量（`--b3-theme-*`）
- **自动适配**：背景色、文字色、边框色自动跟随当前主题
- **对比度优化**：确保在深色背景下代码清晰可读
- **阴影调整**：深色主题下使用更深的阴影效果

### 支持的主题
- ✅ 浅色主题（Light）
- ✅ 深色主题（Dark）
- ✅ 自定义主题（使用标准 CSS 变量）

## 注意事项

1. 样式更改会立即应用到所有代码块
2. 所有风格都会自动适配当前主题（浅色/深色）
3. 建议在保存前先预览效果
4. 重置功能会恢复到默认风格
5. 切换主题时，代码块样式会自动调整
