# 视频加密功能

## 功能概述

视频管理器新增批量加密功能，支持对视频文件进行压缩加密。加密后的视频文件以 `.sn`（单重压缩）或 `.sn2`（双重压缩）格式保存，播放时自动解密，无需输入密码。

## 主要特性

### 1. 批量加密

- 点击顶部工具栏的"批量加密"按钮
- 自动扫描所有未加密的视频文件
- 支持选择单重或双重压缩方式
- 显示实时加密进度

### 2. 加密选项

- **单重压缩**（默认）：生成 `.sn` 格式，速度快
- **双重压缩**：生成 `.sn2` 格式，体积更小但速度较慢

### 3. 自动处理

- 加密完成后自动删除原始视频文件
- 加密视频显示 🔒 标识（单重）或 🔒🔒 标识（双重）
- 播放时自动解密，无需密码

## 加密流程

1. 打开视频管理器（快捷键：Ctrl+Alt+V）
2. 点击工具栏的"批量加密"按钮
3. 选择压缩方式（单重/双重）
4. 点击"开始加密"
5. 系统自动完成：
   - 扫描所有未加密视频
   - 逐个压缩视频文件
   - 保存为 `.sn` 或 `.sn2` 格式
   - 删除原始文件
   - 显示加密进度

## 播放加密视频

1. 加密视频在列表中显示 🔒 或 🔒🔒 标识
2. 直接点击播放，无需输入密码
3. 系统自动解密并播放
4. 解密过程完全在内存中完成

## 技术实现

### 压缩算法

- **压缩算法**：GZIP（CompressionStream API）
- **单重压缩**：压缩一次，速度快
- **双重压缩**：压缩两次，体积更小

### 文件格式

加密文件结构：
```
[1字节头部][压缩数据]
```

头部标识：
- `0x01`：单重压缩（.sn）
- `0x02`：双重压缩（.sn2）

### 核心模块

- `src/features/video/crypto.ts`：压缩/解压核心逻辑
- `src/features/video/index.ts`：视频文件处理 API
- `src/features/video/VideoManager.vue`：UI 界面

## 批量解密

### 解密流程

1. 打开视频管理器
2. 点击工具栏的"批量解密"按钮
3. 确认解密操作
4. 系统自动完成：
   - 扫描所有加密视频
   - 逐个解压视频文件
   - 恢复为原始格式（.mp4 等）
   - 删除加密文件
   - 显示解密进度

### 解密用途

- **平台迁移**：将视频迁移到其他平台或设备
- **备份恢复**：恢复原始视频文件
- **格式转换**：需要使用其他工具处理视频时

## 优势

- **无密码管理**：无需记忆密码，自动解密
- **节省空间**：压缩后文件体积显著减小
- **批量处理**：一键加密/解密所有视频
- **自动清理**：加密/解密后自动删除原文件
- **透明播放**：播放体验与普通视频无异
- **便于迁移**：支持批量解密，方便迁移到其他平台

## 注意事项

- 加密/解密过程中请勿关闭窗口
- 加密后原始文件会被删除，请确认后操作
- 解密后加密文件会被删除，请确认后操作
- 双重压缩适合存储空间有限的场景
- 加密视频仅支持在思源笔记内播放
- 解密后的视频可在任何平台播放
