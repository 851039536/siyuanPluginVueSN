间距 Spacing（全局：@/variables.scss · 数字后缀）
变量CSS 变量值用途
$spacing-1--spacing-14px紧密间距（icon 间距、微型 gap）
$spacing-2--spacing-28px元素内间距（按钮 padding、小 gap）
$spacing-3--spacing-312px中等间距（卡片 padding、列表 gap）
$spacing-4--spacing-416px标准 section 内边距
$spacing-5--spacing-520px大间距
$spacing-6--spacing-624px特大间距（header 水平 padding 上限）
注意：使用 $spacing-1~$spacing-4（数字后缀），不是 $spacing-xs~$spacing-lg。
