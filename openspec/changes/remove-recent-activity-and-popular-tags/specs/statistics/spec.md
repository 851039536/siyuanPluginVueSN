## REMOVED Requirements

### Requirement: 显示热门标签
统计面板应显示使用频率最高的标签云。

**原因**：简化界面，移除不常用的功能，让用户更专注于核心统计数据。

#### Scenario: 查看热门标签
- **当** 用户在统计面板中时
- **那么** 系统应显示使用频率最高的标签云，标签大小根据使用频率动态调整

### Requirement: 显示最近活跃文档
统计面板应显示最近修改过的文档列表。

**原因**：简化界面，移除不常用的功能，让用户更专注于核心统计数据。

#### Scenario: 查看最近活跃文档
- **当** 用户在统计面板中时
- **那么** 系统应显示最近修改过的文档列表
- **并且** 点击文档可以打开该文档

## MODIFIED Requirements

### Requirement: 统计数据展示
统计面板 SHALL 展示用户的工作区统计数据。

#### Scenario: 查看基本统计
- **当** 用户打开统计面板时
- **那么** 系统 SHALL 显示基本的文档统计信息（笔记总数、字数、块数、资源数、标签数、反向链接数）

#### Scenario: 查看今日活动
- **当** 用户查看统计面板时
- **那么** 系统 SHALL 显示今日创建和修改的文档数量

#### Scenario: 查看统计图表
- **当** 用户选择不同的视图模式（日、周、月、年）时
- **那么** 系统 SHALL 显示对应时间段的字数统计图表
