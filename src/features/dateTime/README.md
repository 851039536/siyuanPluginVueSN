# 日期时间插入功能

## 功能说明

提供快捷的斜杠命令，可以快速插入当前的日期、时间或日期时间。

## 使用方法

在编辑器中输入 `/` 后，可以使用以下命令：

### 插入完整日期时间

- `/xz` 或 `/现在` 或 `/xianzai` - 插入当前日期时间（中文）
- `/now` 或 `/datetime` - 插入当前日期时间（英文）

**格式**: `2024-11-24 18:59:41`

### 插入时间

- `/sj` 或 `/时间` 或 `/shijian` - 插入当前时间（中文）
- `/time` - 插入当前时间（英文）

**格式**: `18:59:41`

### 插入日期

- `/rq` 或 `/日期` 或 `/riqi` - 插入当前日期（中文）
- `/jt` 或 `/今天` 或 `/jintian` - 插入今天日期（中文）
- `/date` - 插入当前日期（英文）
- `/today` - 插入今天日期（英文）

**格式**: `2024-11-24`

## 配置

可以在插件设置中启用或禁用此功能：

```typescript
enableDateTime: boolean  // 是否启用日期时间插入功能
```

## 技术实现

- 使用思源笔记的 `protyleSlash` API 注册斜杠命令
- 支持中英文双语命令
- 自动格式化日期时间为标准格式
- 轻量级实现，无外部依赖
