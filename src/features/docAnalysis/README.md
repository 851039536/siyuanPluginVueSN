# 文档分析

分析思源笔记中的文档内容，帮助用户找到小文档/废弃文档，支持多平台发布。以右侧边栏 Dock 面板呈现。

## 主要功能

- **统计分析**：按字数、更新时间、深度、书签等维度统计文档分布
- **多平台发布**：支持 GitHub、Halo、WordPress、Cnblogs 等多种平台
- **书签管理**：通过书签标记文档状态（待发布/已发布/不使用/分类标签）
- **筛选排序**：支持标题/全文搜索、字数范围、多字段排序
- **批量操作**：批量标记、批量发布当前筛选结果

## 书签发布标准

| 书签 | 含义 |
|------|------|
| 待发布 | 调整好的文章，等待发布 |
| 已发布 | 发布完成后重新设书签分类，如：已发布→C# |
| 无 | 无需操作 |
| 不使用 | 废弃文档（归档） |
| 其他（JS、C#、API 等） | 已发布并完成分类处理 |

## 快捷键

`Ctrl+Alt+D` 打开面板
