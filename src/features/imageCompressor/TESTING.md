## 🚀 使用流程

### 1. 启用功能

在插件设置中开启"图片压缩功能"：
1. 点击思源笔记的插件设置图标
2. 找到"启用图片压缩功能"开关
3. 打开开关并保存

### 2. 打开图片压缩器

有两种方式：
- **点击顶部工具栏**: 右上角的图片图标
- **使用快捷键**: `Ctrl + Alt + C`

### 3. 扫描图片

1. 点击"扫描图片"按钮
2. 等待扫描完成(会显示进度条)
3. 扫描会递归检查 `data/assets` 目录下的所有图片

### 4. 选择要压缩的图片

- **单选**: 点击图片卡片
- **全选**: 点击工具栏的"全选"按钮
- **取消全选**: 点击"取消全选"按钮
- 已选择的图片会有蓝色边框高亮

### 5. 配置压缩参数

点击"压缩"按钮后，会弹出配置对话框：

- **压缩质量** (0.1-1.0): 建议 0.8 (80%)
  - 越低压缩率越高，但质量下降
  - 80% 可获得良好的平衡

- **最大文件大小** (0.1-10 MB): 默认 1 MB
  - 超过此大小的图片将被压缩

- **最大宽高** (500-4000 px): 默认 1920 px
  - 超过此尺寸的图片将被等比缩放

- **Web Worker**: 默认开启
  - 在后台线程处理，不阻塞界面

### 6. 执行压缩

1. 确认配置后点击"压缩"
2. 等待压缩完成(显示实时进度)
3. 查看压缩结果统计

### 7. 查看压缩结果

压缩完成后会显示统计信息：
- 总图片数
- 成功/失败数量
- 平均压缩率
- 总共节省的空间 (MB)

### 8. 替换原图

⚠️ **重要提示**: 此操作不可撤销！

1. 确认压缩结果满意后
2. 点击"替换原图"按钮
3. 确认操作
4. 等待替换完成

替换完成后会自动重新扫描，显示最新的图片列表。

## ⚠️ 注意事项

### 文件安全

1. **备份重要图片**: 虽然可以配置备份功能，但建议手动备份
2. **不可撤销**: 替换操作无法撤销，请谨慎操作
3. **引用安全**: 文件名不变，思源笔记的引用不会失效

### 已知限制

1. **GIF 动图**: 压缩后会变成静态图片
2. **透明 PNG**: 转换为 JPEG 会丢失透明度
3. **内存限制**: 一次处理数千张图片可能导致内存压力

### 性能建议

1. **分批处理**: 建议每次处理不超过 100 张图片
2. **开启 Web Worker**: 保持此选项开启
3. **调整配置**: 根据需求调整压缩参数

## 🐛 问题排查

### 问题 1: 扫描不到图片

**可能原因**:
- assets 目录为空
- 权限问题
- 路径配置错误

**解决方案**:
- 检查 `data/assets` 目录是否存在
- 查看控制台错误日志
- 确认思源笔记正在运行

### 问题 2: 压缩失败

**可能原因**:
- 图片损坏
- 格式不支持
- 内存不足

**解决方案**:
- 查看控制台错误信息
- 尝试单张压缩测试
- 降低批处理数量

### 问题 3: 替换失败

**可能原因**:
- 文件被占用
- 权限不足
- 磁盘空间不足

**解决方案**:
- 关闭正在使用该图片的程序
- 检查磁盘空间
- 查看错误提示

### 问题 4: 界面卡顿

**可能原因**:
- 图片数量太多
- 未开启 Web Worker
- 浏览器性能限制

**解决方案**:
- 分批处理
- 开启 Web Worker
- 减少同时处理的图片数量
