1. 搜索快捷键
   ─────────────────────────────────────────────────────────
   
   位置：面板顶部搜索框
   
   功能：
   • 实时搜索快捷键名称、描述、快捷键组合
   • 大小写不敏感
   • 输入后立即过滤显示匹配结果
   
   示例：
   输入 "编" → 显示包含"编"字的所有快捷键
   输入 "Ctrl+A" → 显示快捷键为 Ctrl+A 的快捷键

2. 分类切换
   ─────────────────────────────────────────────────────────
   
   位置：搜索框下方标签栏
   
   分类：
   • 全部 - 显示所有快捷键
   • 思源笔记 - 思源笔记内置快捷键
   • 插件快捷键 - 当前插件的快捷键
   • 自定义 - 用户添加的快捷键
   
   特点：
   • 点击标签即刻切换分类
   • 搜索结果会同时受分类影响
   • 每个分类下的快捷键按功能分组

3. 添加快捷键
   ─────────────────────────────────────────────────────────
   
   入口：面板顶部 [+] 按钮
   
   步骤：
   1. 点击 [+] 按钮，打开"添加快捷键"对话框
   2. 填写必填项：快捷键名称、快捷键
   3. 可选：填写描述和分组
   4. 点击 [确认] 提交
   
   字段说明：
   
   快捷键名称（必填）
   • 快捷键的显示名称
   • 示例："打开搜索"、"快速插入"
   
   描述（可选）
   • 快捷键的功能描述
   • 示例："打开全局搜索面板"
   
   快捷键（必填）
   • 快捷键组合，用 + 分隔
   • 示例："Ctrl+K"、"Cmd+Shift+P"
   • 支持的修饰符：Ctrl, Shift, Alt, Cmd
   
   分组（可选）
   • 快捷键所属分组
   • 默认值："自定义"
   • 示例："编辑操作"、"我的功能"
   
   验证规则：
   • 快捷键名称和快捷键键不能为空
   • 提交前系统会检查必填项
   
4. 删除快捷键
   ─────────────────────────────────────────────────────────
   
   入口：快捷键行右侧 [×] 按钮（仅自定义快捷键可见）
   
   步骤：
   1. 找到要删除的快捷键
   2. 点击行右侧的红色 [×] 按钮
   3. 确认删除（弹出确认对话框）
   4. 快捷键立即从列表移除
   
   说明：
   • 思源笔记和插件快捷键无法删除
   • 删除后无法恢复，请谨慎操作
   • 自定义快捷键行会显示删除按钮

场景 1: 查找思源笔记的撤销快捷键
─────────────────────────────────────────────────────────────

1. 点击右侧边栏 ? 图标打开快捷键面板
2. 点击"思源笔记"标签
3. 在搜索框输入"撤"或"undo"
4. 查看结果：撤销 - Ctrl+Z
   
   OR
   
1. 在搜索框直接输入"Ctrl+Z"
2. 查看匹配的快捷键信息

场景 2: 为自己的功能添加快捷键
─────────────────────────────────────────────────────────────

1. 点击快捷键面板顶部 [+] 按钮
2. 在对话框中填写：
   - 快捷键名称：快速发送
   - 描述：一键发送当前内容
   - 快捷键：Ctrl+Shift+S
   - 分组：我的工具
3. 点击 [确认] 按钮
4. 快捷键立即添加，在"自定义"分类可见

场景 3: 查看所有编辑相关快捷键
─────────────────────────────────────────────────────────────

1. 打开快捷键面板
2. 在搜索框输入"编"或"edit"
3. 面板会同时显示思源笔记和插件中的编辑快捷键
4. 或按分类（"思源笔记"/"插件"）分别查看

场景 4: 整理自己的快捷键
─────────────────────────────────────────────────────────────

1. 在快捷键面板点击"自定义"标签
2. 查看所有已添加的快捷键
3. 对于不需要的快捷键，点击其行右侧的 [×] 删除
4. 对于新功能，点击 [+] 添加快捷键

推荐做法：
─────────────────────────────────────────────────────────────

1. 使用标准修饰符
   ✓ Ctrl+K       (Windows/Linux 标准)
   ✓ Cmd+K        (macOS 标准)
   ✗ Control+K    (错误用法)
   ✗ Command+K    (错误用法)

2. 修饰符顺序
   ✓ Ctrl+Shift+K
   ✓ Ctrl+Alt+K
   ✗ Shift+Ctrl+K (不推荐，但可用)

3. 避免冲突
   • 尽量避免与思源笔记内置快捷键冲突
   • 对于组织内的快捷键，使用统一前缀
   • 示例：Ctrl+Shift 系列作为自定义快捷键

4. 清晰命名
   ✓ 打开搜索 (清晰明了)
   ✓ 插入表格 (动宾结构)
   ✗ S (过于简洁)
   ✗ 快捷键1 (无意义)

Q1: 添加的快捷键会被保存吗？
A: 当前版本的自定义快捷键在会话期间有效。关闭思源笔记后，
   自定义快捷键会丢失。未来版本会支持持久化存储。

Q2: 可以修改现有快捷键吗？
A: 当前版本不支持直接修改。解决方案是：
   1. 删除旧快捷键
   2. 添加新的快捷键

Q3: 快捷键是否真的生效？
A: 快捷键面板仅展示快捷键信息。实际快捷键的执行取决于
   插件本身的快捷键绑定。请参考各功能模块的文档。

Q4: 是否支持导出快捷键？
A: 当前版本不支持导出。后续版本会加入导出为 JSON 的功能。

Q5: 搜索时不区分大小写吗？
A: 对。搜索完全不区分大小写，所有输入都会转为小写后匹配。

Q6: 删除快捷键后能否恢复？
A: 无法恢复。建议删除前仔细确认。

Q7: 可以添加相同的快捷键吗？
A: 可以。系统不会检测重复。但实际执行时，哪个快捷键生效
   取决于插件的实现方式。

Q8: 如何给快捷键分组？
A: 在"分组"字段填入分组名称。相同分组名的快捷键会自动
   归在一起显示。

快捷键面板内的键盘支持：
─────────────────────────────────────────────────────────────

搜索框
  • Ctrl+A      - 全选文本
  • Ctrl+F/Cmd+F - 聚焦搜索框

标签页
  • 可点击切换分类

表单（添加快捷键对话框）
  • Tab         - 切换到下一个字段
  • Shift+Tab   - 切换到上一个字段
  • Enter       - 提交表单（有时）

按钮
  • 可通过 Tab 键聚焦后按 Enter 触发

对于大量快捷键的情况：
─────────────────────────────────────────────────────────────

1. 合理使用分组
   • 按功能模块分组
   • 按使用频率分组
   • 便于查找和管理

2. 有效利用搜索
   • 不需要滚动，直接搜索
   • 快速定位相关快捷键

3. 定期审查
   • 定期删除不使用的快捷键
   • 保持快捷键列表的精简性

4. 文档维护
   • 为复杂快捷键提供详细描述
   • 便于他人理解和协作
